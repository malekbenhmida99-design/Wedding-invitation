# 💍 Malek & Rihem Wedding Invitation

Beautiful bilingual wedding invitation website in English and Arabic.

## Features

✨ **Bilingual Support** - English and Arabic with easy language toggle
🎨 **Gold & White Theme** - Elegant and modern design
📱 **Fully Responsive** - Works on desktop, tablet, and mobile
💌 **RSVP Form** - Collect guest responses with details
🗺️ **Google Maps Integration** - Direct link to venue location
⚡ **Fast Loading** - Optimized for performance

## How to Use

1. Open `index.html` in your browser or visit the GitHub Pages link
2. Click the language toggle button to switch between English and Arabic
3. Share the invitation link with your guests
4. Guests can fill out the RSVP form
